本界主要人物
东巴宇宙观
东巴文化认为宇宙分为三界： 🌑 鬼界（冥界）：居于宇宙最底层，是恶鬼、厉鬼等存在的地方。 🌍 人间：居中，是人类与自然万物共存的世界，人与动植物平等，皆有灵魂。 ☀️ 天界：居于最高处，是神灵、祖先英灵归宿之地。 这三界由《神路图》连接，东巴法师在超度仪式中诵读此经，为亡灵指引升天之路。
