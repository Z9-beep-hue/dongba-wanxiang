## 🌄 项目简介

> *"东巴文是全世界唯一还在使用的象形文字。"*

**东巴·万象**是一款以**纳西族东巴文化**为主题的微信小程序，致力于通过数字化手段保护和传承这一珍贵的非物质文化遗产。

项目名称取自纳西族古老的宇宙观——**"万象"**，寓意东巴文化包罗万象、生生不息。

### ✨ 核心功能

| 模块 | 功能 | 说明 |
|------|------|------|
| 🏠 **首页** | 文化概览 | 东巴文化介绍、数据亮点、每日传承语录 |
| 📖 **象形字典** | 文字图鉴 | 按分类浏览东巴象形文字，查看释义与文化背景 |
| ✍️ **手写体验** | 互动书写 | 在画布上临摹东巴文字，支持笔触颜色/粗细调节 |
| 🔮 **东巴占卜** | 卜卦转盘 | 体验传统东巴三十二卦占卜，获取卦象解读 |
| 🛤️ **神路图传说** | 三界漫游 | 探索东巴神路图中的"神界·人间·鬼界"三界传说 |

## 🛠 技术栈

- **框架**：微信小程序原生开发
- **语言**：JavaScript / WXML / WXSS
- **渲染**：Canvas 2D 手写画布、Canvas 占卜转盘
- **交互**：触摸手写、转盘动画、页面过渡
- **设计**：纳西族传统配色（古铜金 + 深褐 + 赭红）

## 🎨 设计理念

### 配色方案

取自纳西族传统服饰与壁画：

| 颜色 | 色值 | 寓意 |
|------|------|------|
| 🟤 深褐 | `#1C140F` | 纳西族木楞房的深色木纹 |
| 🟡 古铜金 | `#D4A017` | 东巴经书封面的鎏金装饰 |
| 🔴 赭红 | `#C0392B` | 东巴祭祀仪式中的圣火 |
| ⚪ 米白 | `#F5EBD2` | 东巴手工纸的天然色泽 |

### 设计特色

- 🏔 **沉浸式纳西美学**：整体风格融入纳西族传统视觉元素
- 📜 **古朴质感**：模拟东巴手工纸与经卷的质感纹理
- 🎯 **简洁交互**：符合现代小程序设计规范，操作直觉化
- 📱 **全机型适配**：支持不同屏幕尺寸的自适应布局

### 前置要求

1. [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 微信小程序 AppID（测试号或正式号均可）

1. **克隆项目**

   - 点击「编译」即可预览

## 🌍 文化背景

### 东巴文化

**东巴文化**是纳西族独有的一种古老文化，以**东巴象形文字**、**东巴经**和**东巴教**为核心。

- 📝 **东巴文**：全世界唯一仍在使用的象形文字，被誉为"活着的象形文字"
- 📜 **东巴经**：用东巴文写成的宗教经典，记录了纳西族的历史、哲学、天文、医学
- 🏔️ **发源地**：云南省丽江市，世界文化遗产——丽江古城

### 项目意义

东巴文化的传承正面临严峻挑战：
- 掌握东巴文的**东巴祭司**已不足百人
- 年轻一代对东巴文化的了解日益减少
- 大量东巴经书面临自然损毁

**东巴·万象**希望通过数字化的方式，让更多人了解、接触并传承这份珍贵的文化遗产。
